Transcripción Dispositivos Móviles

¡Saludos!

Les doy la bienvenida al módulo 4 con el tema “Dispositivos Móviles”.

Para comenzar, se mencionan, como método de seguridad, las herramientas de gestión de dispositivos móviles, con las que se puede controlar:

  • Autorizaciones de dispositivos.

  • Aplicaciones instaladas.

  • Configuraciones de seguridad en el dispositivo, Wi-Fi y VPN.

  • Manipulaciones indebidas en terminales.

  • Bloqueo remoto.

  • Formateo remoto.

  • Detección de malware.

  • Fortaleza y renovación de contraseñas.

Por otra parte, la protección de información abarca ciertos aspectos, entre ellos:

  • Aplicaciones instaladas.

  • Almacenamiento en la nube.

  • Cifrado del dispositivo.

  • Copias de seguridad.

  • Almacenamiento local de datos empresariales.

Algunos casos de uso de dispositivos a nivel corporativo son:

  • Dispositivo brindado por la empresa para uso corporativo total y personal limitado: En este caso, la empresa tiene cierto control sobre la configuración del hardware y el software para el uso del correo electrónico empresarial, el acceso a la VPN y otras tareas.

  • Dispositivo brindado por la empresa para uso corporativo de alta seguridad: Estos dispositivos requieren un alto grado de control sobre la configuración y software limitado para manejar o utilizar información sensible. Debe estar conectado a la red empresarial para acceder a los servicios corporativos cuando la normativa lo permita.

  • Dispositivo de la persona empleada para uso corporativo (BYOD): La organización proporciona recursos de acceso a la red corporativa y sus servicios. Además, implementa controles de seguridad para mitigar incidentes.
